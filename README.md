# Phone searching

개요
-------------
모바일 기기 분실을 대비한 모바일 위치 추적 앱입니다.
미리 설정한 명령어를 분실한 모바일 기기에 전송하여 해당 기기의 위치정보를 SMS로 받아볼 수 있습니다.
명령어를 통해 분실 기기의 배터리 잔량을 확인하고 배터리 절약 모드로 전환할 수 있습니다.

+ JAVA
+ Android Studio
+ GPS제어 및 주소변환
+ SMS송수신
+ SMS내용 검사를 통해 명령어 판별 
+ 배터리 잔량 확인 및 배터리 절약모드
